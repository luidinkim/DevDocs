# 시작하기

**DevDocs 프로젝트를 로컬 환경에서 실행하는 방법**

## 요구사항

<Callout type="warning">
Node.js 20.0.0 이상이 필요합니다
</Callout>

## 설치 및 실행

<Steps>
### 저장소 클론

```bash
git clone https://github.com/luidinkim/DevDocs.git
cd DevDocs
```

### 패키지 설치

<Tabs items={['npm', 'yarn', 'pnpm']}>
  <Tabs.Tab>
    ```bash
    npm install
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```bash
    yarn install
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```bash
    pnpm install
    ```
  </Tabs.Tab>
</Tabs>

### 개발 서버 시작

```bash
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000) 접속

</Steps>

## 주요 명령어

### 개발
```bash
npm run dev        # 개발 서버 실행
npm run build      # 프로덕션 빌드
npm run start      # 프로덕션 서버 실행
```

### 배포
```bash
npm run build      # 빌드
npm run deploy     # AWS Amplify 배포
```

## 문서 작성

### MDX 파일 생성
`pages` 디렉토리에 `.mdx` 파일을 생성하면 자동으로 라우팅됩니다.

### 컴포넌트 사용
```mdx
import { Callout, Cards, Steps } from 'nextra/components'

<Callout type="info">
  정보 메시지
</Callout>
```

### 메타데이터 설정
각 디렉토리의 `_meta.js` 파일로 사이드바 구조를 정의합니다.
