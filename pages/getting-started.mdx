# 시작하기

DevDocs 문서 플랫폼에 오신 것을 환영합니다. 이 가이드는 플랫폼을 효과적으로 탐색하고 활용하는 방법을 안내합니다.

## 네비게이션 가이드

### 사이드바 네비게이션
왼쪽 사이드바를 통해 모든 문서 섹션에 접근할 수 있습니다. 각 섹션은 논리적으로 구성되어 있으며, 클릭하면 하위 항목이 확장됩니다.

### 검색 기능
상단의 검색창을 사용하여 필요한 정보를 빠르게 찾을 수 있습니다. 검색은 제목, 내용, 코드 블록을 모두 포함합니다.

### 테마 전환
오른쪽 상단의 테마 토글 버튼으로 라이트/다크 모드를 전환할 수 있습니다.

## 콘텐츠 구조

### 문서 유형

1. **튜토리얼** - 단계별 학습 가이드
2. **가이드** - 종합적인 주제 설명
3. **레퍼런스** - API 및 기술 참조
4. **개념** - 이론적 배경 설명
5. **How-to** - 특정 작업 수행 방법

### 난이도 표시

- 🟢 **초급** - 기초 개념과 시작 가이드
- 🟡 **중급** - 실무 적용 가능한 내용
- 🔴 **고급** - 심화 주제와 최적화
- ⚫ **전문가** - 아키텍처와 설계 패턴

## 유용한 기능

### 코드 복사
모든 코드 블록에는 복사 버튼이 있어 쉽게 코드를 복사할 수 있습니다.

```javascript
// 예제 코드 - 복사 버튼을 클릭해보세요
function hello() {
  console.log("Hello, DevDocs!");
}
```

### 목차 네비게이션
오른쪽의 목차(Table of Contents)를 통해 현재 페이지 내에서 빠르게 이동할 수 있습니다.

### 이전/다음 네비게이션
페이지 하단의 이전/다음 버튼으로 순차적으로 문서를 탐색할 수 있습니다.
