# 시작하기

DevDocs 문서 플랫폼을 시작하는 방법을 안내합니다.

<Callout type="info">
  **Nextra v3** 기반으로 구축된 이 문서 사이트는 강력한 검색 기능과 다국어 지원을 제공합니다.
</Callout>

## 빠른 시작

<Steps>
### 프로젝트 클론
 
```bash
git clone https://github.com/luidinkim/DevDocs.git
cd DevDocs
```

### 의존성 설치

<Tabs items={['npm', 'yarn', 'pnpm']}>
  <Tabs.Tab>
    ```bash
    npm install
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```bash
    yarn install
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```bash
    pnpm install
    ```
  </Tabs.Tab>
</Tabs>

### 개발 서버 실행

```bash
npm run dev
```

### 브라우저에서 확인

http://localhost:3000 으로 접속하세요.
</Steps>

- **Deployment**: AWS Amplify
