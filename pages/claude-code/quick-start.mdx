</Tabs>

### 1.2 API 키 설정

```bash

export CLAUDE_API_KEY="your-api-key-here"
```

## Step 2: 첫 번째 프로젝트

### 2.1 새 프로젝트 생성

```bash
# React 프로젝트 생성
claude-code new my-app --template react

# 프로젝트 디렉토리로 이동
cd my-app

# Claude Code 실행
claude-code
```

### 2.2 첫 명령어 실행

```bash
# 프로젝트 구조 분석
/analyze

# 간단한 컴포넌트 생성
/implement Button 컴포넌트를 만들어줘
```

## Step 3: SuperClaude 활성화

### 3.1 SuperClaude 설치

```bash
# SuperClaude 프레임워크 다운로드
git clone https://github.com/anthropics/superclaude ~/.claude

# 자동 활성화
claude-code config --enable-superclaude
```

### 3.2 프로젝트별 설정

프로젝트 루트에 `CLAUDE.md` 파일 생성:

```markdown
# 프로젝트 설정

## 기술 스택
- Frontend: React 18 + TypeScript
- Styling: Tailwind CSS
- State: Zustand
- Testing: Jest + React Testing Library

## 코딩 컨벤션
- 컴포넌트: PascalCase
- 함수: camelCase
- 파일명: kebab-case

## 자동 실행
- 커밋 전: npm run lint
- 푸시 전: npm test
```

## 자주 묻는 질문

### Q: 에러가 발생하면 어떻게 하나요?

```bash
# 디버그 모드 활성화
claude-code --debug

# 로그 확인
claude-code logs

# 도움말
claude-code help
```

### Q: 오프라인에서도 사용 가능한가요?

일부 기능은 오프라인 사용 가능하지만, 전체 기능을 위해서는 인터넷 연결이 필요합니다.

### Q: 팀에서 사용하려면?

팀 라이센스를 구매하고 공유 설정 파일을 사용하세요:

```bash
# 팀 설정 공유
claude-code team init
claude-code team share CLAUDE.md
```
