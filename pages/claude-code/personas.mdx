## 페르소나 활성화 방법

### 자동 활성화
Claude Code가 작업 내용을 분석하여 자동으로 적합한 페르소나를 활성화합니다.

### 수동 활성화
```bash
# 단일 페르소나
/implement API --persona-backend

# 복합 페르소나
/analyze --persona-architect --persona-performance
```

## 🏗️ Architect (아키텍트)

### 역할
시스템 전체를 조망하는 설계자. 장기적 관점에서 확장 가능한 아키텍처를 설계합니다.

### 전문 분야
- 시스템 아키텍처 설계
- 마이크로서비스 설계
- 디자인 패턴 적용
- 기술 스택 선택
- 확장성 계획

### 활용 예시
```bash
# 시스템 전체 설계
/design 이커머스 플랫폼 --persona-architect --ultrathink

# 아키텍처 리뷰
/analyze --persona-architect --focus architecture
```

## 🎨 Frontend (프론트엔드)

### 역할
사용자 경험을 최우선으로 하는 UI/UX 전문가. 접근성과 성능을 고려한 인터페이스를 구현합니다.

### 전문 분야
- React/Vue/Angular 개발
- 반응형 디자인
- 웹 접근성 (WCAG 2.1)
- 성능 최적화
- 크로스 브라우저 호환성

### 활용 예시
```bash
# 컴포넌트 개발
/implement 대시보드 UI --persona-frontend

# 접근성 개선
/improve --persona-frontend --focus accessibility
```

## ⚙️ Backend (백엔드)

### 역할
안정적이고 확장 가능한 서버 시스템을 구축하는 전문가. 데이터 무결성과 보안을 최우선으로 합니다.

### 전문 분야
- RESTful API 설계
- 데이터베이스 최적화
- 인증/인가 시스템
- 마이크로서비스
- 메시지 큐

### 활용 예시
```bash
# API 개발
/implement REST API --persona-backend

# 데이터베이스 최적화
/improve database --persona-backend --focus performance
```

## 🔒 Security (보안)

### 역할
시스템의 취약점을 찾아 보호하는 보안 전문가. OWASP 가이드라인을 준수합니다.

### 전문 분야
- 취약점 분석
- 보안 감사
- 암호화 구현
- 인증 시스템
- GDPR 준수

### 활용 예시
```bash
# 보안 감사
/analyze --persona-security --ultrathink

# 취약점 수정
/improve --persona-security --safe-mode
```

## ⚡ Performance (성능)

### 역할
시스템 성능을 극한까지 최적화하는 전문가. 측정 가능한 개선을 추구합니다.

### 전문 분야
- 성능 프로파일링
- 병목 지점 분석
- 캐싱 전략
- 로드 밸런싱
- 리소스 최적화

### 활용 예시
```bash
# 성능 분석
/analyze --persona-performance --measure-first

# 최적화 적용
/improve --persona-performance --benchmark
```

## 🔍 Analyzer (분석가)

### 역할
복잡한 문제의 근본 원인을 찾는 탐정. 체계적이고 논리적인 접근을 합니다.

### 전문 분야
- 근본 원인 분석
- 디버깅
- 로그 분석
- 성능 분석
- 코드 리뷰

### 활용 예시
```bash
# 버그 원인 분석
/troubleshoot "메모리 누수" --persona-analyzer

# 시스템 분석
/analyze --persona-analyzer --think-hard
```

## ✅ QA (품질 보증)

### 역할
제품 품질을 보장하는 테스트 전문가. 엣지 케이스를 찾아내는 것이 특기입니다.

### 전문 분야
- 테스트 전략 수립
- 자동화 테스트
- 엣지 케이스 발견
- 회귀 테스트
- 성능 테스트

### 활용 예시
```bash
# 테스트 작성
/test --persona-qa --comprehensive

# 품질 검증
/analyze --persona-qa --focus quality
```

## 🔧 Refactorer (리팩토러)

### 역할
코드 품질과 유지보수성을 개선하는 전문가. 기술 부채를 체계적으로 해결합니다.

### 전문 분야
- 코드 리팩토링
- 디자인 패턴
- 기술 부채 관리
- 코드 스멜 제거
- 테스트 커버리지

### 활용 예시
```bash
# 코드 개선
/improve --persona-refactorer

# 기술 부채 해결
/cleanup --persona-refactorer
```

## 🚀 DevOps (데브옵스)

### 역할
개발과 운영을 통합하는 전문가. 자동화와 모니터링을 통해 안정적인 배포를 보장합니다.

### 전문 분야
- CI/CD 파이프라인
- 인프라 자동화
- 컨테이너화
- 모니터링
- 로그 관리

### 활용 예시
```bash
# CI/CD 구축
/implement CI/CD pipeline --persona-devops

# 인프라 자동화
/build infrastructure --persona-devops
```

## 📚 Mentor (멘토)

### 역할
지식을 전달하고 성장을 돕는 교육자. 복잡한 개념을 쉽게 설명합니다.

### 전문 분야
- 코드 리뷰
- 교육 자료 작성
- 베스트 프랙티스
- 페어 프로그래밍
- 지식 전달

### 활용 예시
```bash
# 개념 설명
/explain React hooks --persona-mentor

# 교육 문서 작성
/document tutorial --persona-mentor
```

## ✍️ Scribe (문서 작성자)

### 역할
명확하고 체계적인 문서를 작성하는 전문가. 다국어 지원과 문화적 맥락을 고려합니다.

### 전문 분야
- 기술 문서 작성
- API 문서화
- 사용자 가이드
- 릴리스 노트
- 다국어 번역

### 활용 예시
```bash
# 문서 작성
/document --persona-scribe

# 다국어 문서
/document --persona-scribe=ko
```
