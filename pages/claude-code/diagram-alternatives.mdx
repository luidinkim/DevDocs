# 📊 문서용 다이어그램 도구 비교

<Callout type="info" emoji="💡">
Mermaid 외에도 MDX/Nextra에서 사용할 수 있는 훌륭한 시각화 도구들이 많습니다!
</Callout>

## 🎨 주요 대안들

### Excalidraw ✏️
**손그림 스타일의 협업 다이어그램**
- 친근하고 부담없는 스케치 스타일
- 실시간 협업 지원
- Mermaid → Excalidraw 변환 가능
- 웹사이트: excalidraw.com

### D2 (Declarative Diagramming) 🔷
**선언적 다이어그램 언어**
- 서버사이드 렌더링
- 브라우저 의존성 없음
- 빠른 SVG 생성
- 웹사이트: d2lang.com

### Markmap 🧠
**마크다운 → 마인드맵**
- 자동 마인드맵 생성
- 인터랙티브 확장/축소
- 마크다운 문법 그대로 사용
- 웹사이트: markmap.js.org

### PlantUML 🌱
**엔터프라이즈급 다이어그램**
- UML 다이어그램 전문
- 복잡한 시스템 설계
- 다양한 다이어그램 타입
- 웹사이트: plantuml.com
