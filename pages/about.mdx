# 소개

## DevDocs 프로젝트

DevDocs는 개발 지식을 체계적으로 정리하고 공유하기 위한 문서 플랫폼입니다.

### 목적

- **지식 관리** - 개인 개발 경험과 학습 내용을 체계적으로 정리
- **교육 자료** - 학생들과 주니어 개발자를 위한 학습 리소스 제공
- **커뮤니티 기여** - 오픈소스 커뮤니티와 지식 공유

### 기술 스택

- **프레임워크**: Next.js 14 + Nextra 3
- **언어**: TypeScript, React
- **스타일링**: Tailwind CSS
- **배포**: AWS Amplify
- **버전 관리**: Git & GitHub

### 주요 기능

#### 문서 관리
- Markdown/MDX 기반 작성
- 버전 관리 및 히스토리 추적
- 다국어 지원 (한국어/영어)

#### 사용자 경험
- 반응형 디자인
- 다크/라이트 테마
- 전문 검색 기능
- 코드 하이라이팅

#### 성능 최적화
- 정적 사이트 생성 (SSG)
- 이미지 최적화
- 캐싱 전략
- CDN 배포
