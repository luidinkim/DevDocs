## 핵심 기능

- **🔍 강력한 검색** - FlexSearch 기반 실시간 문서 검색
- **🌙 다크 모드** - 시스템 설정 연동 자동 테마 전환
- **📱 반응형 디자인** - 모바일부터 데스크톱까지 최적화
- **⚡ 빠른 성능** - Next.js 15 기반 최적화된 렌더링

## 기술 스택

- **Framework**: Next.js 15 + Nextra 3
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Deployment**: AWS Amplify
