# Claude Code 마스터 가이드

## AI 페어 프로그래밍의 새로운 기준

Claude Code는 단순한 코드 생성 도구가 아닙니다. 프로젝트의 컨텍스트를 이해하고, 아키텍처를 설계하며, 
프로덕션 레벨 코드를 작성하는 진정한 AI 개발 파트너입니다.

<Callout type="info">
**핵심 가치**: Claude Code로 3개월 프로젝트를 2주 만에 완성하세요.
실제 스타트업들이 MVP를 10배 빠르게 출시하고 있습니다.
</Callout>

## 왜 Claude Code인가

### 실제 파일 시스템 접근
ChatGPT와 달리 실제 파일을 읽고, 수정하고, 생성합니다. 복사-붙여넣기는 이제 과거의 일입니다.

### 200K 토큰 컨텍스트
전체 코드베이스를 한 번에 이해합니다. 수백 개 파일의 프로젝트도 문제없습니다.

### SuperClaude 프레임워크
11개 전문가 페르소나와 MCP 서버 통합으로 엔터프라이즈급 개발이 가능합니다.

## SuperClaude: 게임 체인저

### sc: 명령어 체계

### 필수 MCP 서버

#### Sequential Thinking
복잡한 문제를 체계적으로 분해하고 해결합니다.
```bash
sc:troubleshoot "메모리 누수" --seq
# 1. 증상 분석 → 2. 원인 파악 → 3. 해결책 도출 → 4. 구현
```

#### Context7
1000+ 라이브러리의 최신 문서를 즉시 참조합니다.
```bash
sc:implement "React 18 Suspense" --c7
# React 18 공식 문서 기반 정확한 구현
```
