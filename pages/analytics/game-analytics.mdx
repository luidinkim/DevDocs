# 게임 분석 도구 비교

게임 개발과 운영에 필수적인 분석 도구들을 비교합니다.

## Unity Analytics

### 특징
- Unity 엔진과 완벽한 통합
- 실시간 이벤트 추적
- 커스텀 이벤트 지원
- 무료 티어 제공

### 주요 메트릭
- DAU/MAU (일간/월간 활성 사용자)
- 세션 길이와 빈도
- 리텐션 율
- 수익화 지표

### 구현 예제
```csharp
using UnityEngine.Analytics;

// 커스텀 이벤트 전송
Analytics.CustomEvent("levelComplete", new Dictionary<string, object>
{
    { "level", currentLevel },
    { "score", playerScore },
    { "time", completionTime }
});
```

## GameAnalytics

### 특징
- 크로스 플랫폼 지원
- 자동 이벤트 추적
- 실시간 대시보드
- 무료로 시작 가능

### 핵심 기능
- 플레이어 진행도 추적
- 에러 및 크래시 리포팅
- A/B 테스팅
- 커스텀 디멘션

### 구현 예제
```csharp
// SDK 초기화
GameAnalytics.Initialize();

// 진행도 이벤트
GameAnalytics.NewProgressionEvent(
    GAProgressionStatus.Complete,
    "world_01",
    "level_01"
);
```

## Firebase Analytics

### 특징
- Google 생태계 통합
- 무제한 이벤트 추적
- 예측 분석
- BigQuery 연동

### 고급 기능
- 사용자 속성 관리
- 오디언스 세그먼테이션
- 원격 구성(Remote Config)
- 크래시리틱스 통합

### 구현 예제
```javascript
// 이벤트 로깅
firebase.analytics().logEvent('level_complete', {
  level: 5,
  score: 10000,
  character: 'warrior'
});
```
