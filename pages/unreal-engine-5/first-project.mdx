### 프로젝트 설정

상세 설정 옵션들:

| 설정 | 추천값 | 설명 |
|------|--------|------|
| **Target Platform** | Desktop | 개발 대상 플랫폼 |
| **Quality Preset** | Maximum | 그래픽 품질 (나중에 조정 가능) |
| **Starter Content** | ✅ Include | 학습용 기본 에셋 포함 |
| **Raytracing** | ❌ Disable | RTX 카드 필요, 성능 영향 |

<Callout type="info">
  **Starter Content**는 다양한 머티리얼, 메시, 파티클 등 학습에 유용한 에셋들을 포함합니다. 초보자는 반드시 포함시키세요.
</Callout>

## 첫 실행 및 테스트

### 인터페이스 확인

프로젝트가 열리면 다음을 확인할 수 있습니다:

- **뷰포트**: 기본 레벨이 표시됨
- **콘텐츠 브라우저**: StarterContent 폴더
- **월드 아웃라이너**: 레벨의 액터들
- **디테일 패널**: 선택한 액터 속성

### 플레이 테스트

## 프로젝트 구조 이해하기

### 폴더 구조

```
MyFirstUE5Project/
├── Config/           # 프로젝트 설정 파일
├── Content/          # 게임 에셋 (중요!)
│   ├── StarterContent/
│   └── ThirdPerson/
├── Intermediate/     # 임시 파일 (무시)
├── Saved/           # 저장된 데이터
└── MyFirstUE5Project.uproject  # 프로젝트 파일
```

### 중요 폴더 설명

| 폴더 | 용도 | 백업 필요 |
|------|------|-----------|
| **Content** | 모든 게임 에셋 | ✅ 필수 |
| **Config** | 프로젝트 설정 | ✅ 필수 |
| **Source** | C++ 소스 코드 | ✅ (C++ 사용시) |
| **Intermediate** | 캐시 파일 | ❌ |
| **Saved** | 로그, 백업 | ⚠️ 선택적 |

## 프로젝트 저장

<Callout type="error" emoji="⚠️">
  작업 중 자주 저장하는 습관을 들이세요!
</Callout>

### 저장 단축키

- **Ctrl + S**: 현재 레벨 저장
- **Ctrl + Shift + S**: 모든 변경사항 저장
- **Ctrl + Alt + S**: 다른 이름으로 저장

### 자동 저장 설정

1. **Edit** → **Editor Preferences**
2. **Loading & Saving** → **Auto Save**
3. 다음 옵션 활성화:
   - Enable AutoSave ✅
   - Save interval: 10분
   - Save Maps ✅
   - Save Content ✅

## 문제 해결

### 자주 발생하는 문제들

<Tabs items={['실행 오류', '성능 문제', '저장 실패']}>
  <Tabs.Tab>
    **"Failed to open project"**
    - 프로젝트 경로에 한글/특수문자 확인
    - 관리자 권한으로 실행
    - 엔진 버전 확인
    
    **"Shader compiling"**
    - 첫 실행시 정상 (5-20분 소요)
    - 백그라운드에서 진행됨
  </Tabs.Tab>
  <Tabs.Tab>
    **낮은 FPS**
    - Scalability Settings 조정
    - Engine Scalability → Medium 또는 Low
    
    **메모리 부족**
    - 최소 16GB RAM 권장
    - 다른 프로그램 종료
  </Tabs.Tab>
  <Tabs.Tab>
    **"Failed to save"**
    - 디스크 공간 확인
    - 파일 권한 확인
    - 안티바이러스 예외 추가
  </Tabs.Tab>
</Tabs>
