### 타이머 디스플레이

<Code language="blueprint">
Widget Structure:
├── Horizontal Box
│   ├── Icon (Clock Image)
│   ├── Time Text (Bind)
│   └── Best Time (Smaller)
└── Progress Bar (Optional)

Binding Function: GetFormattedTime
├── Get Game Mode
├── Get Current Game Time
├── Format: MM:SS.MS
├── Color Code:
│   ├── Green: < 50% of limit
│   ├── Yellow: 50-80% of limit
│   └── Red: > 80% of limit
└── Return Formatted String
</Code>

## 🔄 UI 애니메이션

### 위젯 애니메이션

<Steps>

### 애니메이션 생성

1. Widget Designer에서 **Animations** 탭
2. **+ Animation** 클릭
3. 타임라인에서 키프레임 설정

### 공통 애니메이션

```blueprint
Fade In/Out:
├── Opacity: 0 → 1
├── Duration: 0.3s
└── Curve: Ease In/Out

Scale Bounce:
├── Scale: 1 → 1.2 → 1
├── Duration: 0.5s
└── Curve: Elastic

Slide In:
├── Position: Off-screen → Target
├── Duration: 0.4s
└── Curve: Ease Out
```

</Steps>

### 트랜지션 효과
