### 5. AI 시스템

#### Behavior Tree 구조

```
Root
├─ Selector (우선순위)
│  ├─ Sequence (전투)
│  │  ├─ Check Enemy in Range
│  │  ├─ Aim at Enemy
│  │  └─ Attack
│  ├─ Sequence (추적)
│  │  ├─ Find Player
│  │  └─ Move to Player
│  └─ Patrol (기본)
```

#### AI 구성 요소

- **Behavior Tree**: AI 로직
- **Blackboard**: 데이터 저장
- **AI Controller**: 실행 제어
- **EQS**: 환경 쿼리 시스템
- **Navigation Mesh**: 경로 탐색
