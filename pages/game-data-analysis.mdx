## 핵심 메트릭

### 사용자 메트릭

<Tabs items={['획득', '활성화', '리텐션', '수익화']}>
  <Tabs.Tab>
    - **DAU (Daily Active Users)**: 일간 활성 사용자
    - **MAU (Monthly Active Users)**: 월간 활성 사용자
    - **신규 사용자**: 일별/주별/월별 신규 유저
    - **설치수**: 앱 설치 횟수
  </Tabs.Tab>
  
  <Tabs.Tab>
    - **세션 길이**: 평균 플레이 시간
    - **세션 빈도**: 일일 평균 세션 수
    - **튜토리얼 완료율**: 온보딩 성공률
    - **첫 구매까지 시간**: FTUE 효과성
  </Tabs.Tab>
  
  <Tabs.Tab>
    - **D1/D7/D30 리텐션**: 1일/7일/30일 후 복귀율
    - **이탈률 (Churn Rate)**: 사용자 이탈 비율
    - **LTV (Lifetime Value)**: 사용자 생애 가치
    - **Stickiness**: DAU/MAU 비율
  </Tabs.Tab>
  
  <Tabs.Tab>
    - **ARPU**: 사용자당 평균 수익
    - **ARPPU**: 과금 사용자당 평균 수익
    - **Conversion Rate**: 무료→과금 전환율
    - **IAP 수익**: 인앱 구매 수익
  </Tabs.Tab>
</Tabs>

### 게임플레이 메트릭

- **레벨 진행도**: 각 레벨별 완료율과 이탈률
- **아이템 사용 패턴**: 인기 아이템과 사용 빈도
- **소셜 기능 활용**: 친구 초대, 길드 가입률
- **이벤트 참여율**: 특별 이벤트 참여도

## 주요 분석 도구 비교

| 도구 | 장점 | 단점 | 적합한 경우 |
|------|------|------|------------|
| **GameAnalytics** | 무료, 사용 쉬움 | 고급 기능 제한 | 인디/중소 개발사 |
| **Unity Analytics** | Unity 통합 우수 | Unity 전용 | Unity 프로젝트 |
| **Firebase** | Google 생태계 | 복잡한 설정 | 크로스 플랫폼 |
| **Amplitude** | 강력한 분석 | 비용 높음 | 대규모 프로젝트 |
| **mixpanel** | 실시간 분석 | 학습 곡선 | 라이브 서비스 |

<Callout type="warning">
  분석 도구 선택 시 데이터 프라이버시 규정(GDPR, COPPA 등)을 반드시 고려하세요.
</Callout>

## 분석 사례 연구

### 성공 사례: 리텐션 40% 개선

<Callout type="success">
  **A게임**은 데이터 분석을 통해 튜토리얼 이탈 구간을 발견하고 개선하여 D7 리텐션을 25%에서 35%로 향상시켰습니다.
</Callout>

**핵심 개선 사항:**
1. 튜토리얼 3단계 난이도 하향 조정
2. 첫 보상 시점을 5분에서 2분으로 단축
3. 소셜 기능 노출 시점 최적화

### 실패 사례: 과도한 데이터 수집

<Callout type="error">
  **B게임**은 너무 많은 이벤트를 추적하여 분석 비용이 급증하고 인사이트 도출이 어려워졌습니다.
</Callout>

**교훈:**
- 핵심 KPI에 집중
- 단계적 이벤트 추가
- 정기적인 이벤트 정리
