### Unity Analytics

#### 개요
Unity 엔진과 완벽하게 통합된 분석 솔루션입니다.

- **통합 수준**: Unity 에디터 내장
- **실시간 성능**: 우수
- **학습 곡선**: 낮음

#### 장점
- ✅ **Unity 통합**: 설정 없이 바로 사용
- ✅ **Unity 서비스 연동**: Ads, IAP 등과 연계
- ✅ **자동 이벤트**: 기본 이벤트 자동 수집

#### 단점
- ❌ **Unity 전용**: 다른 엔진 사용 불가
- ❌ **가격 정책**: MAU 기반 과금
- ❌ **제한적 커스터마이징**: Unity 생태계 내 제한

#### 가격
- **Personal**: 무료 (50K MAU까지)
- **Plus**: $40/월 (200K MAU)
- **Pro**: 맞춤 가격

#### 적합한 경우
- Unity 기반 프로젝트
- Unity 생태계 활용
- 중소 규모 모바일 게임

### Firebase Analytics

#### 개요
Google의 강력한 모바일 앱 플랫폼의 일부입니다.

- **Google 통합**: AdMob, AdWords 연동
- **무제한 이벤트**: 500개 이벤트 타입
- **실시간 분석**: 지원

#### 장점
- ✅ **무료 기본 기능**: 대부분 기능 무료
- ✅ **Google 생태계**: 광고, A/B 테스트 통합
- ✅ **BigQuery 연동**: 고급 분석 가능
- ✅ **크로스 플랫폼**: iOS, Android, Web

#### 단점
- ❌ **복잡한 설정**: 초기 설정 난이도 높음
- ❌ **모바일 중심**: PC/콘솔 지원 제한
- ❌ **학습 곡선**: Google Cloud 이해 필요

#### 가격
- **Spark**: 무료 (기본 기능)
- **Blaze**: 사용량 기반 과금

#### 적합한 경우
- 모바일 중심 프로젝트
- Google 광고 활용
- 대규모 데이터 분석 필요

### Amplitude

#### 개요
엔터프라이즈급 제품 분석 플랫폼입니다.

- **고급 분석**: 코호트, 퍼널, 리텐션
- **행동 분석**: 사용자 여정 추적
- **예측 분석**: ML 기반 인사이트

#### 장점
- ✅ **강력한 분석**: 업계 최고 수준
- ✅ **실시간 데이터**: 즉각적인 인사이트
- ✅ **팀 협업**: 대시보드 공유
- ✅ **A/B 테스트**: 내장 실험 기능

#### 단점
- ❌ **높은 비용**: 스타트업에 부담
- ❌ **복잡성**: 전문가 필요
- ❌ **오버스펙**: 소규모 프로젝트

#### 가격
- **Starter**: 무료 (10만 이벤트/월)
- **Growth**: $995/월
- **Enterprise**: 맞춤 가격

#### 적합한 경우
- 대규모 라이브 서비스
- 데이터 팀 보유
- 성장 중인 스타트업

## 플랫폼별 추천

### 모바일 게임
- **소규모**: GameAnalytics
- **중규모**: Firebase + GameAnalytics
- **대규모**: Amplitude + Firebase

### PC/콘솔 게임
- **인디**: GameAnalytics
- **AA**: GameAnalytics + 커스텀 솔루션
- **AAA**: 자체 구축 또는 Amplitude

### 크로스 플랫폼
- **Unity 기반**: Unity Analytics + GameAnalytics
- **Unreal 기반**: GameAnalytics + 커스텀
- **자체 엔진**: Firebase + Amplitude

## 단계별 도구 선택 가이드

<Steps>
### 프로젝트 규모 확인
- 예상 MAU < 10K → 소규모
- 10K < MAU < 100K → 중규모
- MAU > 100K → 대규모

### 플랫폼 결정
- 모바일 전용?
- PC/콘솔 포함?
- 웹 지원 필요?

### 예산 검토
- 무료만 가능 → GameAnalytics
- 월 $100 이하 → Firebase/Unity Analytics
- 월 $1000+ → Amplitude/mixpanel

### 팀 역량 평가
- 데이터 분석가 있음?
- 예 → **Amplitude/BigQuery**
- 아니오 → **GameAnalytics/Unity Analytics**

### 성장 가능성
- 빠른 성장 예상?
- 예 → **확장 가능한 솔루션**
- 아니오 → **간단한 솔루션**

### 최종 결정
- 실시간 분석 중요?
- 예 → **Amplitude/mixpanel**
- 아니오 → **GameAnalytics**
</Steps>

## 자주 묻는 질문

### Q: GameAnalytics가 정말 완전 무료인가요?
A: 네, GameAnalytics는 모든 기능을 무료로 제공합니다. 숨겨진 비용이 없습니다.

### Q: 여러 분석 도구를 동시에 사용해도 되나요?
A: 가능하지만 성능 영향과 데이터 일관성을 고려해야 합니다. 2개 이하를 권장합니다.

### Q: 언제 유료 도구를 고려해야 하나요?
A: MAU 10만 이상, 고급 분석 필요, 전담 데이터 팀이 있을 때 고려하세요.

### Q: GDPR 규정은 어떻게 준수하나요?
A: 선택한 도구의 GDPR 가이드를 따르고, 사용자 동의를 받으며, 데이터 삭제 요청에 대응해야 합니다.
