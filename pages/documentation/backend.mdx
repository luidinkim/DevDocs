# 백엔드 개발

확장 가능하고 안정적인 서버 애플리케이션 구축 가이드

## Node.js

### Express.js
- 라우팅과 미들웨어
- 에러 처리
- 인증과 권한
- API 설계

### NestJS
- 의존성 주입
- 모듈 시스템
- 가드와 인터셉터
- 마이크로서비스

## Python

### FastAPI
- 타입 힌트와 자동 문서화
- 비동기 프로그래밍
- 의존성 주입
- 보안과 인증

### Django
- MVC 패턴
- ORM과 마이그레이션
- Admin 인터페이스
- REST Framework

## API 설계

### RESTful API
- HTTP 메서드 활용
- 상태 코드 전략
- 페이지네이션
- 버전 관리

### GraphQL
- 스키마 설계
- 리졸버 구현
- 성능 최적화
- N+1 문제 해결

## 마이크로서비스

### 아키텍처 패턴
- API Gateway
- Service Mesh
- Event Sourcing
- CQRS

### 통신 패턴
- 동기 vs 비동기
- Message Queue
- gRPC
- WebSocket
