# 데이터베이스

데이터 저장, 관리, 최적화를 위한 종합 가이드

## 관계형 데이터베이스 (RDBMS)

### PostgreSQL
- 고급 쿼리 최적화
- 인덱싱 전략
- 파티셔닝
- JSON 데이터 타입

### MySQL
- 스토리지 엔진
- 레플리케이션
- 샤딩
- 성능 튜닝

## NoSQL 데이터베이스

### MongoDB
- 도큐먼트 모델링
- 집계 파이프라인
- 샤딩과 레플리카셋
- 트랜잭션

### Redis
- 데이터 구조
- 캐싱 전략
- Pub/Sub
- 클러스터링

- 1NF, 2NF, 3NF

### 인덱싱

### 쿼리 최적화
